UI Controls Xamarin Forms (Android, iOS, UWP) to give productivity to your code.

The component contains a versatile and easy-to-use image rating, giving dynamism to your application, as well as being able to use it with numbers and letters natively. Swap Image along with Swap Label makes it easy to replace texts and images without reloading the page.

Lean more

Rating

Easy, fast and dynamic.
Easily create a rating using any image of your choice, indicating pre-selection image and post-selection image. You can create it with the size you need using the amount of elements you want.

Swap

Usability.
Easily swap images with only one xaml tag, indicating the first and second images, with a simple event and you can control the functionality you need.

< ContentPage xmlns ="http://xamarin.com/schemas/2014/forms"
xmlns : x ="http://schemas.microsoft.com/winfx/2009/xaml"
xmlns : local ="clr-namespace:RatingStar"
xmlns : RAS ="clr-namespace:Laavor;assembly=RatingAndSwap"
xmlns : Class ="RatingStar.MainPage">
< StackLayout >

<!-- The image needs to be within the project that is running as the default, Android, iOS, UWP. It is of your choice you can use either one -->
< RAS : RatingImage x:Name="stars" ItemsNumber="7" Padding="40" ImageDeselect="starWhite.png" ImageSelect="starYellow.png" ImageWidth="30" ImageHeight="30" OnSelect="RatingImage_OnSelect" >


<-- The InitialValue gives you the dynamics of already leaving a default value selected in case the user does not click -->
< RAS : RatingImage x:Name="balls" InitialValue="3" ItemsNumber="5" Padding="10" ImageDeselect="ball1.png" ImageSelect="ball2.png" ImageWidth="30" ImageHeight="30" >


<-- The property IsReadOnly causes you to not be able to change the rating value, just displays -->
< RAS : RatingImage x:Name="search" IsReadOnly="True" InitialValue="4" ItemsNumber="8" Padding="10" ImageDeselect="search1.png" ImageSelect="search2.png" ImageWidth="25" ImageHeight="25" >

< / StackLayout >
< / ContentPage >

< ContentPage xmlns ="http://xamarin.com/schemas/2014/forms"
xmlns : x ="http://schemas.microsoft.com/winfx/2009/xaml"
xmlns : local ="clr-namespace:NumberRating"
xmlns : RAS ="clr-namespace:Laavor;assembly=RatingAndSwap"
xmlns : Class ="NumberRating.MainPage">
< StackLayout >

<!-- You can choose text and background colors for selected and unselected items -->
< RAS : RatingNumber ItemsNumber="7" NumberFontSize="30" Padding="10" TextColorSelect="White" TextColorDeselect="Black" BackgroundColorSelect="Black" BackgroundColorDeselect="White" >


<-- If you set BlockSelect to true, you will not be able to change the rating value after the first click. -->
< RAS : RatingNumber ItemsNumber="5" NumberFontSize="30" Padding="10" TextColorSelect="White" TextColorDeselect="Black" BackgroundColorSelect="Black" BackgroundColorDeselect="White" BlockSelect="True" >


<-- The property IsReadOnly causes you to not be able to change the rating value, just displays -->
< RAS : RatingNumber x:Name="ratingLamp" IsReadOnly="True" ItemsNumber="8" Padding="20" ImageDeselect="lamp1.png" ImageSelect="lamp2.png" ImageWidth="35" ImageHeight="35" >


<-- In this mixed example the top lamps will light when the respective number underneath is selected -->
< RAS : RatingNumber x:Name="numberLamp" ItemsNumber="8" NumberFontSize="25" Padding="25" TextColorSelect="White" TextColorDeselect="Gold" BackgroundColorSelect="Gold" BackgroundColorDeselect="White" OnSelect="RatingNumber_OnSelect" >

< / StackLayout >
< / ContentPage >
